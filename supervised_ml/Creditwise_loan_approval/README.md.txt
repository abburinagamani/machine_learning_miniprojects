# Creditwise Loan Approval System

## Problem Statement
The goal of this project is to predict whether a loan application should be approved based on applicant information such as income, loan amount, and credit history.

## Dataset
- File name: loan_approval_data.xls
- Type: Supervised classification dataset
- Target variable: Loan_Status

## Approach
- Data preprocessing and missing value handling
- Encoding categorical features
- Feature scaling
- Model building using:
  - Logistic Regression
  - Decision Tree Classifier
- Model evaluation using accuracy score and confusion matrix

## Results
- Achieved around 80% accuracy on test data
- Logistic Regression provided stable and interpretable results

## Tools & Libraries
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib / Seaborn
