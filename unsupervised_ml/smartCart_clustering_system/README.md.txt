# Smart Cart Clustering System

## Problem Statement
The objective of this project is to segment customers into different groups based on their purchasing behavior using unsupervised machine learning techniques.

## Dataset
- File: smart_cart_data.csv
- Type: Unsupervised learning dataset
- Features include purchase frequency, product categories, and spending patterns.

## Approach
- Data cleaning and preprocessing
- Feature scaling
- Clustering using:
  - K-Means Clustering
  - DBSCAN (if applicable)
- Cluster visualization and interpretation

## Results
- Identified distinct customer segments
- Useful for targeted marketing and recommendation strategies

## Tools & Libraries
- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib / Seaborn
